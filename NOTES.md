# Moonlander 開発メモ

## ゲーム展開ロードマップ

### Stage 1: 月面着陸 ✅
- 宇宙船を操作してデブリを避けながらゴールに着陸
- 水平＋低速で着地するとクリア

### Stage 2: 宇宙飛行士降下 ✅
- 着陸成功後、宇宙飛行士がハシゴを降りてくる演出
- `player-f / player-l / player-r` アセット

### Stage 3: 地上戦アクション ✅
- 左右移動＋ジャンプ（押し時間で強弱）
- スペースでビーム発射（向きに追従・3段階威力）
- ビーム残量ゲージ、宇宙船の下でチャージ
- 地底人（alien）が地面からランダム出現→主人公を追撃
- 5体撃破ごとにボス出現（大型・HP高・低速）
- 接触で爆発ゲームオーバー

### Stage 4: クルー救出ミッション（次に実装）
- 岩陰から 2〜3 人の仲間を救出
- 仲間は岩オブジェクトの陰に隠れている（発見要素）
- 接触で連れ歩き、宇宙船に戻すと救出カウント
- 連れている間は移動速度/ビーム威力にペナルティ（検討）
- 全員救出でステージクリア

#### 必要アセット
- 岩オブジェクト（隠れ場所として配置）
- 壊れた大きい月面探索車（ランドマーク／背景演出）

### Stage 5: 母艦帰還ミッション
- 月面脱出後、母艦ステーションへ戻る
- 宇宙飛行シーケンス（Stage 1 の要素を応用？）

### Stage 6: 母艦での謎解きパート
- 到着後はアクションから謎解きアドベンチャーへ転換
- 月で集めた情報／仲間の証言を元にステーション内を調査
- 月の先住種族と失踪事件の真相に迫る

## 世界観メモ

- 月の地下に潜む先住種族と、行方不明になっていた仲間
- 救出した仲間の台詞で月の謎が断片的に明かされる演出を検討
- 最終的にはステーションで全てが繋がる謎解きに収束

### インスピレーション（全部入り）

#### 映画・小説（世界観・演出）
- **『星を継ぐもの』** — 月で見つかる古代の謎、科学的考察
- **『プロジェクト・ヘイルメアリー』** — 孤独な宇宙飛行士、異星知性との交流、パズル解明
- **『エイリアン』** — 地底から湧く脅威、閉所の恐怖
- **『2001年宇宙の旅』** — 静謐で荘厳な空気、謎めいたモノリス的存在
- **『月に囚われた男』(Moon 2009)** — 月面基地の孤独、アイデンティティの謎
- **『ゼロ・グラビティ』** — 無重力サバイバル、帰還への執念
- **『インターステラー』** — 宇宙スケールの謎、時間・次元の歪み
- **『エリジウム』** — 軌道ステーションの格差・閉鎖空間

#### ゲーム（操作感・モーション）
- **Another World** — ロトスコープ風の滑らかで重みのあるアニメ、映画的な演出、静寂と緊張
- **Prince of Persia (初代)** — 慣性のある歩き／走り／ジャンプ、フレームごとに意味がある動き

→ Stage 3 以降の宇宙飛行士モーションは、チカチカしたドット絵ではなく**重量感のある滑らかな動き**を目指す。ジャンプ・着地・走り・しゃがみなどフレーム数を多めに。

## 既知の不具合 / 技術メモ

### 8bit PostFX パイプラインと GeometryMask の干渉
- `PixelArt` PostFX パイプライン (index.html 冒頭の WebGL シェーダー) を `cameras.main.setPostPipeline('PixelArt')` で適用すると、エイリアン捕獲時の「地中へ消える」演出が壊れる（astronaut が地面下でも見えたまま）。
- 原因: Phaser 3.55.2 では PostFX の render target に stencil buffer が正しく引き継がれず、`createGeometryMask()` の clipping が効かなくなる。
- 試した対策と結果:
  - `createBitmapMask()` に変更 → NG（同じ症状）。
  - 捕獲時に `removePostPipeline` で一時停止 → NG（フリーズ or 効かず）。
  - astronaut の alpha フェードで代替 → 視覚的に違和感。
- 暫定対応: `setPostPipeline('PixelArt')` を **コメントアウト** して 8bit を OFF。
- 恒久対応候補:
  1. Phaser 3.60+ へアップグレード（PostFX と mask の互換性改善あり）。
  2. 8bit を CSS filter（`image-rendering: pixelated` ＋ 低解像度レンダリング）で再現して PostFX を使わない。
  3. マスクを使わずに astronaut を alpha/crop で消す方式に寄せる。
- 関連: `stargate.html`（動作確認用の単独シェーダーサンドボックス、git 管理外）。
